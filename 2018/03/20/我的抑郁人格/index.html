我的抑郁人格

红手印压上的那一刻，我明白，这一切终于结束了，新的生活开始了……我不由得开始思考，这四个月的经历是我到现在为止的人生浓墨重彩的一笔，让我深刻的思考我究竟是怎样一个人，为什么我是这样的，这样的我好在哪里？不足又是什么？未来的我又该如何去选择？

好朋友有一次聊天和我说见过三种类型的富二代，第一种是成天瞎晃悠、不正经的事总是少不了他，这个很明显就是钱太多了成为放纵自己的理由；第二种是一天也没啥卵事，该吃吃该喝喝该学习学习，没有压力随性而为，这种人呢由于知道没有后顾自忧所以自己想干什么就干什么，再该拼的年纪也没必要把自己累着；那么第三种人呢，低调到你根本看不出来他又什么背景，反而一天要死要活的、拼命的干“正事”，让人看了真的是想不明白也“自愧不如”，其实我就是他眼里的这种人。但是很可惜的是，他想不明白，我此前也想不明白，直到我了解了我这种抑郁人格的产生的原因我才恍然大悟。

一个人是否开心，和他的财富、地位、家庭、外表基本上没什么关系。

此前我就有一个疑问，为什么我无法真正的开心起来，至少我自己是这样认为的。原来开心真的和你有多少钱、什么背景、长的好看不好看没什么关系。那到底是什么在影响我们的心情————心理结构。

心理结构每人都有，它影响了我们如何看待别人、自己以及我与周围世界的关系。
它决定我们选择什么样的工作、朋友、伴侣、建立什么样的家庭，甚至小到微信头像选什么。
在面对同样一种挑战的时候，人们会有不同的情绪，由此一个人要处理自己的情绪，就会发展出来一系列的行为模式来保护自己，我们把它叫做某一种人格。

当然人的心理结构都不是单一的，一个人不是某一种「人格」可以定义的，只是总有一种人格作为主导，作为你处理挑战的主导行为模式，而我因对外界的模型便是一个抑郁人格的处理模式，他是怎样的呢？

抑郁人格有几大特征悲观、性情忧郁、特别善于自我贬低。

面对事物总会主动或者下意识的认为它会朝着不好的方向发展，导致多愁善感，进一步就会觉得这里不行了，那里也不行，从而丧失自我，丢了信心。当然这种人格遇到不好的事物是这样的，但我们知道大多数的事物本身并不会呈现出一种消极或者不好的结果，而是一种未知状态，你不知道它最终是好还是坏。从某种意义上来说，抑郁人格的个体不会倾向于像好的结果发展，因此他们会开始存在焦虑，开始否定现在不能掌控事物的自己，对自己的不接纳，反而成了鞭策他们不断向上努力的动力。

相反，我觉得自己挺好的，我不需要奋斗来证明自己的有价值的，我们可能就少了一个内在驱动力。

所以很多时候，事物仅仅是一个正常的健康的发展历程，而我却感觉到压力山大，其实多数时候的压力都是自己给自己的，所以我不是不满足，而是这种抑郁人格给你带来的“焦虑”再告诉你：不，我没办法确定这是好的，所以我必须马不停蹄的做，直到我确定它是好的为止。
这里值得一提的是，心理学家克莱因指出抑郁人格的心里过程的两个位置：

PS 位置（偏执分裂位）

D 位置（抑郁位）

PS位置的人只会感觉到烦躁，不会抑郁，最常见的一种做法就是遇到困难就甩锅，把不好的和自己撇的一干二净，然而并不是真的不在意，而是在回避。如果能够这样就不去在意自然是好，然如果没有办法经过一段时间的思考1就会进入第二个位置也就是D位置。

抑郁位是一个思考的位置，在D位之上。
D位是一个思考和产生行动的位置。这是抑郁带来我们的财富和价值。

长久的处于PS位置只会让自己成为一个不负责任的人、消极避世的人，而过于持久的处于D位置则会像我这样，感觉不到真正的开心。那么我的抑郁人格究竟从哪里来？

第一，遗传因素
抑郁有生理遗传因素，还有一些心理遗传因素。原生家庭中没有得到亲密养育者的关注，或者是出生在一个相对不健康的家庭里面，一个孩子很可能在成长过程中没有办法发展出适应性的能力。
第二，分离个体化
心理上没有充分独立的话，他的感受也会很焦虑，我要为父母的开心而负责，我必须按照他们的想法来安排自己的生活。这是抑郁者一个特别核心的特征，就是我的感受不重要，别人的感受才是最重要的。这个过程是非常糟糕的，因为无论如何他都不会快乐，如果他满足了父母的想法，他没有办法感受到成为自己的快乐，如果他违背了父母的意愿，他会陷入深深的自责和内疚当中。
第三，忌讳表达哀伤的家庭氛围
我们不能表达脆弱，不能表达内心真实的感受，这就好像我们在心理种了一颗种子，这颗种子在不断地告诉你，你做的还不够好，你还可以更努力。这是第三个，苛刻的家庭环境。

我并不想过多的去强调遗传因素，因为虽然同年相对缺乏父爱，但我有一个非常快乐的童年，我并不想把症结过多的归咎于感情缺失，但是所谓原生家庭的影响是我无法辩驳的，因为这种共性没办法忽略。更多的我想强调分离个体化，意愿的强烈冲突，我就是源自于我和父亲的相互不了解和而我日益强大的个人思维的矛盾是最主要的原因，我想调解的办法只能是进行沟通了，但我知道很多事情不能强求，双方各退一部，妥协也是一种解决问题的办法。如果说是忌讳表达哀伤的家庭氛围，更不如说是自己不愿意表达、无处表达和不能表达，这可能要源于小时候对父亲强大榜样作用的敬畏，不愿不敢不行，当时就是这种想法了。

其实到这里已经很清楚了，性格、人格的形成时漫长的也是难以更改的，但是我们都知道一种性格或者一种人格绝对不是单方面好或者单方面坏的，就比如我的抑郁人格，在PS位的时候恰到好处能让自己变得简单，但是过了就是烦躁和消极；在D位置的时候获得了一种动力，但长期的保持又成了一种负担。因此不管时在PS位还是D位，更重要的时如何去保持一种平衡，我很享受D位置带给我的动力和激情，弹药如何防范长期处于D位置的焦虑和累积的负担呢？

如何保护自己不受抑郁情绪的伤害 ？
  • [ ] 给自己制定一些非常细节、明确的目标。比如，每周跟一个好朋友一起吃顿饭，每天做一点家务活。可操作的目标，能够让你对生活有更多的控制感和成就感;
  • [ ] 去行动。比如，去楼下便利店买一杯咖啡，研究一下手边的笔记本是什么材质，把你从思维的死结中带出来，回到现实生活中;
  • [ ] 正念冥想。正念从本质上是在训练你的注意力，它能帮助你觉察你的情绪、想法，让你意识到那些抑郁的想法可能不是真实的;
  • [ ] 去运动;
  • [ ] 给自己创造一个情绪上的安全岛。

一个非常简单可操作的事情就是，你给自己列一个名单，名单上的人，是你觉得你跟他表达你的脆弱感，表达你难过或悲伤的时候，他不会来评价你。

当我们遇到困难、压力、痛苦的时候，给自己一定心理上的空间，允许自己体会这些脆弱的感受，其实是最最关键的。

这是文章《人只有在抑郁的时候，才愿意思考》给的答案，我想我的答案更明确也更简单：

  • [x] 非工作日一定要大胆的去拥抱除了工作以外的家庭、朋友、爱好。
  • [x] 一旦陷入抑郁的不良症结，别想着如何返回好的状态，立刻停止纠结的思考，走出当下，在新的环境中让思想重新回归到现实生活。
  • [x] 没有什么是一个五公里解决不了的。
  • [x] 我的情绪安全岛就是我的亲人、朋友和爱人，他们是无可替代的！
